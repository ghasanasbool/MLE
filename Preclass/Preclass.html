Instructions

How to approach the sheet

For each question, try to write the likelihood first. Then take logs. Only then think about differentiating or solving for the maximum.

Reminder: MLE does not start by guessing a formula. It starts with a probability model for the observed data.
SymbolMeaning
L(θ; y)Likelihood: the joint probability/density of the observed data, viewed as a function of θ.
ℓ(θ; y)Log-likelihood: log L(θ; y). It has the same maximiser as L.
θ̂The parameter value chosen by maximum likelihood.
Core questions

Construct the likelihood, then maximise

Question 1 Poisson likelihood

Suppose transaction counts over four equal time windows are:

y = (2, 0, 3, 1)

Assume Yi ~ Poisson(λ), independently.

  1. Write the likelihood L(λ; y).
  2. Write the log-likelihood ℓ(λ; y).
  3. Simplify the log-likelihood as far as possible.
Hint: for a Poisson variable, f(y; λ) = exp(−λ)λy/y!.
Solution: L(λ;y)=∏ exp(−λ)λyᵢ/yᵢ!. Here n=4 and Σyᵢ=6. So ℓ(λ;y)=−4λ+6log(λ)−[log(2!)+log(0!)+log(3!)+log(1!)]. Since log(0!)=log(1!)=0, this is −4λ+6log(λ)−log(2!)−log(3!).

Question 2 Poisson MLE

For general independent Poisson data Y1,…,Yn, show that the MLE of λ is the sample mean.

Hint: differentiate ℓ(λ)=−nλ+(Σyᵢ)log(λ)−constant.
Solution: ∂ℓ/∂λ = −n + (Σyᵢ)/λ. Set equal to zero: −n + (Σyᵢ)/λ = 0, so λ̂ = (Σyᵢ)/n = ȳ. The second derivative is −(Σyᵢ)/λ² < 0 when there is at least one event, confirming a maximum.

Question 3 Poisson numerical estimate

For the sample:

y = (1, 3, 2, 4, 2)

Compute λ̂. Then interpret it in words if these are transaction counts per five-minute window.

Solution: Σyᵢ = 12 and n = 5, so λ̂ = 12/5 = 2.4. Interpretation: the estimated average number of transactions per five-minute window is 2.4.

Question 4 Compare likelihood values

Using the sample y=(1,3,2,4,2), compare the log-likelihood at λ=1 and λ=2.4. You do not need to calculate the factorial constant exactly; explain why it cancels in the comparison.

Hint: use ℓ(λ)=−5λ+12log(λ)−constant.
Solution: Ignoring the same constant, ℓ(1)=−5(1)+12log(1)=−5. ℓ(2.4)=−12+12log(2.4)≈−12+10.506=−1.494. Since −1.494 > −5, λ=2.4 gives the higher log-likelihood. The factorial term depends on the data, not λ, so it is identical for both candidates.

Question 5 Bernoulli likelihood

Suppose Yi ~ Bernoulli(p), independently, and the observed data are:

y = (1, 0, 1, 1, 0)
  1. Write the likelihood.
  2. Find p̂.
Solution: L(p;y)=∏pyᵢ(1−p)1−yᵢ. There are three ones and two zeros, so L(p)=p³(1−p)². The log-likelihood is 3log(p)+2log(1−p). The first-order condition gives 3/p − 2/(1−p)=0, so p̂=3/5=0.6.

Question 6 Normal mean, known variance

Suppose Xi ~ Normal(μ, σ²), independently, and σ² is known. Show that the MLE of μ is the sample mean.

Hint: after dropping constants, maximise −(1/(2σ²))Σ(xᵢ−μ)².
Solution: The log-likelihood is constant − (1/(2σ²))Σ(xᵢ−μ)². Differentiate with respect to μ: ∂ℓ/∂μ = (1/σ²)Σ(xᵢ−μ). Setting this to zero gives Σxᵢ − nμ = 0, so μ̂ = x̄.

Question 7 Normal numerical MLE

For the sample x=(4,5,7), assume Xi ~ Normal(μ, σ²), independently, with both μ and σ² unknown.

  1. Compute μ̂.
  2. Compute σ̂² using the MLE formula.
  3. How is this different from the usual unbiased sample variance?
Solution: μ̂=(4+5+7)/3=16/3≈5.333. The MLE variance is σ̂²=(1/n)Σ(xᵢ−x̄)². The squared deviations are 1.778, 0.111, 2.778, which sum to 4.667. So σ̂²=4.667/3≈1.556. The usual unbiased sample variance divides by n−1, while the Normal MLE divides by n.

Question 8 Why logs?

Explain why maximising L(θ;y) and maximising log L(θ;y) give the same estimate.

Solution: The log function is strictly increasing. If one parameter value gives a larger likelihood than another, it also gives a larger log-likelihood. Therefore the location of the maximum is unchanged. Logs are used because products become sums and derivatives become easier.

Question 9 Score and first-order condition

In words, what does it mean to solve the equation ∂ℓ(θ;y)/∂θ = 0?

Solution: It means finding a point where the log-likelihood stops increasing as θ changes. At an interior maximum, the slope of the log-likelihood is zero. This is a candidate for the MLE, though we still need to check it is a maximum and respects any parameter restrictions.

Question 10 Parameter restrictions

For each parameter below, state the restriction and explain why it matters:

  1. Poisson λ
  2. Bernoulli p
  3. Normal σ
Solution: Poisson λ must be positive because it is an average event rate. Bernoulli p must lie between 0 and 1 because it is a probability. Normal σ must be positive because it is a standard deviation. Numerical optimisation must respect these restrictions; otherwise it may try impossible parameter values.
Applied thinking

Connect the mathematics to financial modelling

Question 11 Returns versus prices

In the coding seminar, students model daily returns rather than the S&P 500 price level. Give two reasons why returns are more natural for the introductory Normal MLE exercise.

Solution: Prices tend to trend and are not centred around a stable mean. Returns are changes in value and often fluctuate around a more stable centre. Returns also have a more interpretable mean and volatility for risk analysis, although they are not perfectly Normal in practice.

Question 12 Model criticism

Suppose the fitted Normal curve is placed over a histogram of daily returns. What visual evidence might suggest that the Normal model is too simple?

Solution: Possible warning signs include heavy tails, skewness, outliers, a sharper central peak than the Normal curve, or clusters of extreme observations. These suggest that the Normal model may underestimate the probability of large movements or fail to capture asymmetry.

Question 13 Conditional likelihood

Suppose income Y depends on years of education X. A simple conditional model is:

Yi | Xi=xi ~ Normal(β0 + β1xi, σ²)

Write the conditional log-likelihood for observations i=1,…,n. You do not need to solve for the parameters.

Hint: each observation has mean μᵢ=β₀+β₁xᵢ.
Solution: ℓ(β₀,β₁,σ;y|x)=Σ[−log(σ)−0.5log(2π)−(1/(2σ²))(yᵢ−β₀−β₁xᵢ)²]. This is conditional because the distribution of Y changes with xᵢ.

Question 14 Exponential waiting times

Suppose waiting times between trades are positive continuous variables and are modelled as Exponential(λ), with density:

f(t; λ) = λ exp(−λt),   t ≥ 0, λ > 0

For independent waiting times t1,…,tn:

  1. Write the likelihood.
  2. Write the log-likelihood.
  3. Find λ̂.
Hint: ℓ(λ)=nlog(λ)−λΣtᵢ.
Solution: L(λ;t)=∏λ exp(−λtᵢ)=λⁿ exp(−λΣtᵢ). The log-likelihood is ℓ(λ)=nlog(λ)−λΣtᵢ. Differentiate: ∂ℓ/∂λ=n/λ−Σtᵢ. Set equal to zero: λ̂=n/Σtᵢ=1/t̄. The estimated event rate is the reciprocal of the average waiting time.
Reflection: Questions 1–14 move from discrete counts, to binary outcomes, to continuous Normal returns, to waiting times. The same basic MLE logic appears in every case.
